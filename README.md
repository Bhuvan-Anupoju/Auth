# 🔐 Auth System
#### 🔰Overview

- A full-stack Authentication System built from scratch with secure user authentication and password recovery functionality. This project includes Sign Up, Sign In, Forgot Password with OTP verification, JWT-based authentication, and email services using Node Mailer.
 ## ✨ Features

-  User Registration (Sign Up)
-  User Login (Sign In)
-  Secure Password Hashing
-  JWT Authentication
-  Forgot Password Feature
-  OTP Verification through Email
-  Password Reset Functionality
-  Protected Routes
-  Responsive User Interface
-  Email Sending using Node Mailer

## 🔄 Project Workflow
### 👤 Sign Up
- User enters username, email, and password
- Password is hashed before storing in database 
- User account is created successfully 
### 🔑 Sign In
- User logs in using email and password
- Credentials are verified 
- JWT token is generated after successful login 
- Token is used for authentication and protected routes 🛡️
### 🔁 Forgot Password
- User enters registered email 📩
- OTP is generated and sent to the user's email using  Node Mailer ✉️
- User verifies OTP 🔢
- User can reset password securely 🔒
